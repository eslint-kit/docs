# base

## Manual installation

```bash
$ yarn add -D @eslint-kit/eslint-config-base
```

## When to use?

Always use this preset.

## Included plugins

* `eslint-plugin-import`
* `eslint-plugin-unicorn`
* `eslint-plugin-sonarjs`

## Included presets

* `plugin:sonarjs/recommended`
